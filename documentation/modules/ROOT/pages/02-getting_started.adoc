== ACS Architecture

The StackRox Kubernetes Security Platform installs as a set of pods in your Kubernetes or OpenShift cluster and includes the following components:

image::architecture_acs.png[ACS Architecture, 800]

* [**Central**](https://help.stackrox.com/docs/system-info/stackrox-architecture/#central): Gathers and displays information from other components (1 for multiple clusters).
* [**Sensor**(https://help.stackrox.com/docs/system-info/stackrox-architecture/#sensor)]: Collects and augments data from the Collector (1 for each cluster.)
* [**Scanner**](https://help.stackrox.com/docs/system-info/stackrox-architecture/#sensor): Scans images for vulnerabilities (1 for multiple clusters)
* [**Collector**](https://help.stackrox.com/docs/system-info/stackrox-architecture/#sensor): Collects and monitors container activities (1 on each node)
* [**Admission controller**](https://help.stackrox.com/docs/system-info/stackrox-architecture/#sensor): Interacts with Kubernetes API server and prevents creating workloads that donâ€™t adhere to security policies	(1 for each cluster)
