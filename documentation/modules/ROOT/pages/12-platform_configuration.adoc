== ACS Secured Clusters

image::multicluster/0_multi.png[ACS Multi 1, 800]

[.console-input]
[source,bash,subs="attributes+,+macros"]	
----	
unzip -d sensor sensor-[cluster-name].zip

./sensor/sensor.sh
----
